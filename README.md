# Data Cleaning & Reporting Automation

## Project Overview
This project demonstrates data cleaning and reporting automation using Python and Pandas. The dataset was cleaned by removing duplicates, handling missing values, formatting phone numbers, and generating an automated report.

## Features
- Read Excel dataset using Pandas
- Removed duplicate records
- Removed unnecessary columns
- Cleaned the Last_Name column
- Cleaned and formatted phone numbers
- Split the Address column into separate fields
- Handled missing values (NaN)
- Standardized customer information
- Generated an automated data cleaning report
- Exported cleaned data to Excel

## Technologies Used
- Python
- Pandas
- Jupyter Notebook

## Output Files
- Customer_Call_List_Cleaned.xlsx
- Data_Cleaning_Report.xlsx
